# 📝 Инструкция по настройке OpenAI API

## 🔑 Где получить API ключ OpenAI

### Шаг 1: Зарегистрируйтесь на OpenAI

1. Перейдите на сайт: https://platform.openai.com/
2. Нажмите "Sign Up" (Регистрация) или "Log In" (Вход), если уже зарегистрированы
3. Войдите в аккаунт

### Шаг 2: Получите API ключ

1. После входа перейдите в раздел **API Keys**: https://platform.openai.com/api-keys
2. Нажмите кнопку **"Create new secret key"** (Создать новый секретный ключ)
3. Придумайте название для ключа (например, "Aldea Tools")
4. Скопируйте ключ (он начинается с `sk-...`)
   ⚠️ **ВАЖНО**: Скопируйте ключ сразу! Вы больше не сможете его увидеть

### Шаг 3: Пополните баланс (если нужно)

1. Перейдите в раздел **Billing**: https://platform.openai.com/account/billing
2. Нажмите **"Add payment method"** (Добавить способ оплаты)
3. Добавьте карту (минимальный депозит $5)

## 🖥️ Настройка для локальной разработки

### Шаг 1: Создайте файл `.env` в корне проекта

Создайте файл `.env` в корне проекта (там же, где `package.json`):

```env
# OpenAI API Configuration
OPENAI_API_KEY=sk-ваш-ключ-здесь
OPENAI_MODEL=gpt-4o-mini

# Database (если еще не настроено)
DATABASE_URL=your_database_url_here
```

### Шаг 2: Вставьте свой ключ

Замените `sk-ваш-ключ-здесь` на реальный ключ, который вы скопировали.

**Пример:**
```env
OPENAI_API_KEY=sk-proj-abc123def456ghi789jkl012mno345pqr678stu901vwx234yz
OPENAI_MODEL=gpt-4o-mini
```

### Шаг 3: Запустите сервер

```bash
npm run server
```

## 🌐 Настройка для Netlify (Production)

### Шаг 1: Войдите в Netlify Dashboard

1. Перейдите на https://app.netlify.com/
2. Выберите ваш сайт

### Шаг 2: Добавьте переменные окружения

1. Перейдите в **Site settings** → **Environment variables**
2. Нажмите **"Add a variable"**
3. Добавьте следующие переменные:

   **Переменная 1:**
   - Key: `OPENAI_API_KEY`
   - Value: `sk-ваш-ключ-здесь` (вставьте ваш ключ)

   **Переменная 2 (опционально):**
   - Key: `OPENAI_MODEL`
   - Value: `gpt-4o-mini` (или другую модель)

4. Нажмите **"Save"**

### Шаг 3: Перезапустите деплой

После добавления переменных:
1. Перейдите в **Deploys**
2. Нажмите **"Trigger deploy"** → **"Clear cache and deploy site"**

## 🤖 Как выбрать модель OpenAI

### Доступные модели:

| Модель | Описание | Цена (за 1M токенов) | Качество | Рекомендация |
|--------|----------|---------------------|----------|--------------|
| `gpt-4o-mini` | Быстрая и дешевая | ~$0.15/$0.60 | Хорошее | ✅ **Рекомендуется** для начала |
| `gpt-4o` | Лучшее качество | ~$2.50/$10.00 | Отличное | Для премиум функций |
| `gpt-4-turbo` | Стабильная версия | ~$10/$30 | Отличное | Альтернатива gpt-4o |

### Как изменить модель:

#### Для локальной разработки:

В файле `.env`:
```env
OPENAI_MODEL=gpt-4o-mini    # Дешевая модель (рекомендуется)
# или
OPENAI_MODEL=gpt-4o         # Лучшее качество (дороже)
```

#### Для Netlify:

1. В Netlify Dashboard → **Site settings** → **Environment variables**
2. Найдите `OPENAI_MODEL` или создайте новую переменную
3. Измените значение на нужную модель
4. Сохраните и перезапустите деплой

## 🔒 Безопасность

⚠️ **НИКОГДА не публикуйте ваш API ключ:**
- ❌ Не добавляйте `.env` в Git
- ❌ Не коммитьте ключ в код
- ❌ Не делитесь ключом публично

✅ **Убедитесь, что `.env` в `.gitignore`:**
Проверьте файл `.gitignore`, там должна быть строка:
```
.env
```

## 🆘 Решение проблем

### Ошибка: "OPENAI_API_KEY is not configured"

**Решение:**
- Проверьте, что файл `.env` создан в корне проекта
- Убедитесь, что ключ введен правильно (начинается с `sk-`)
- Перезапустите сервер

### Ошибка: "Invalid API Key"

**Решение:**
- Проверьте, что ключ скопирован полностью
- Убедитесь, что нет лишних пробелов
- Проверьте баланс на OpenAI аккаунте

### Ошибка: "Insufficient quota"

**Решение:**
- Пополните баланс на OpenAI: https://platform.openai.com/account/billing
- Проверьте лимиты использования
