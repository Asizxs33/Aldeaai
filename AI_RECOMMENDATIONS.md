# Рекомендации по выбору AI моделей для Aldea AI

## Функции сайта, требующие AI:

1. **Чат-бот (Bot.jsx)** - Диалоговый AI ассистент для учителей
2. **Генерация презентаций (Presentation.jsx)** - Структурированный контент для слайдов
3. **Генератор материалов (Generator.jsx)** - Генерация текстовых материалов по предметам
4. **Tulga (Tulga.jsx)** - Генерация текста для видео с историческими персонажами

---

## 🏆 Рекомендации по моделям:

### **Вариант 1: OpenAI GPT-4o / GPT-4o-mini (РЕКОМЕНДУЕТСЯ)**

#### Преимущества:
- ✅ Отличное качество генерации на казахском, русском и английском
- ✅ Поддержка JSON форматирования (важно для презентаций)
- ✅ Хорошо понимает контекст образования
- ✅ Стабильная работа, хорошая документация
- ✅ GPT-4o-mini - очень дешевая альтернатива для простых задач

#### Рекомендации по использованию:

**Чат-бот (Bot.jsx):**
- Модель: `gpt-4o-mini` (дешево) или `gpt-4o` (лучше качество)
- Обоснование: Диалоги не требуют сложной логики, gpt-4o-mini достаточно
- Стоимость: $0.15 / 1M входных токенов, $0.60 / 1M выходных токенов (mini)
- Стоимость: $2.50 / 1M входных токенов, $10.00 / 1M выходных токенов (gpt-4o)

**Презентации (Presentation.jsx):**
- Модель: `gpt-4o` (обязательно для структурированного JSON)
- Обоснование: Нужна высокая точность в генерации JSON структуры
- Стоимость: $2.50 / 1M входных, $10.00 / 1M выходных токенов

**Генератор материалов (Generator.jsx):**
- Модель: `gpt-4o-mini` (достаточно для текстов)
- Стоимость: $0.15 / 1M входных, $0.60 / 1M выходных токенов

**Tulga (Tulga.jsx):**
- Модель: `gpt-4o-mini` (короткие тексты)
- Стоимость: $0.15 / 1M входных, $0.60 / 1M выходных токенов

#### Примерная стоимость в месяц (1000 пользователей):

| Функция | Запросов/мес | Токенов вход | Токенов выход | Стоимость |
|---------|-------------|--------------|---------------|-----------|
| Чат-бот (gpt-4o-mini) | 20,000 | ~500K | ~2M | **$1.35** |
| Презентации (gpt-4o) | 5,000 | ~300K | ~1M | **$11.75** |
| Генератор (gpt-4o-mini) | 10,000 | ~200K | ~800K | **$0.54** |
| Tulga (gpt-4o-mini) | 3,000 | ~100K | ~300K | **$0.21** |
| **ИТОГО** | | | | **~$14/мес** |

*При масштабировании до 10,000 пользователей: ~$140/мес*

### **Вариант 3: Комбинированный подход (ОПТИМАЛЬНЫЙ)**

#### Гибридная стратегия:

1. **Чат-бот, Генератор, Tulga:** `gpt-4o-mini` (дешево, качество достаточное)
2. **Презентации:** `gpt-4o` (нужна точность JSON)

**Почему это оптимально:**
- Экономия до 60% по сравнению с использованием gpt-4o везде
- Высокое качество там, где это критично (JSON в презентациях)
- Достаточное качество для простых задач

**Стоимость: ~$14-18/мес (1000 пользователей)**

## 🎯 Финальная рекомендация:

### **Используйте GPT-4o-mini для большинства задач + GPT-4o для презентаций**

**Почему:**
1. ✅ Лучшее соотношение цена/качество
2. ✅ GPT-4o-mini отлично работает для текстов и диалогов
3. ✅ GPT-4o нужен только для точного JSON в презентациях
4. ✅ Легко масштабируется
5. ✅ Хорошая поддержка казахского языка (лучше чем у большинства конкурентов)

### Реализация:

```javascript
// Для чат-бота, генератора, Tulga
const model = 'gpt-4o-mini';

// Для презентаций (критично для JSON)
const model = 'gpt-4o';
```

## 🔧 Рекомендации по оптимизации:

1. **Кэширование:** Кэшируйте частые запросы (похожие темы)
2. **Rate limiting:** Ограничьте количество запросов на пользователя
3. **Streaming:** Используйте streaming для лучшего UX
4. **Fallback:** Имейте резервную модель (gpt-4o-mini если gpt-4o недоступен)
5. **Мониторинг:** Отслеживайте использование токенов и стоимость
