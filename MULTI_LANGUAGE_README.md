# Многоязычная поддержка / Multi-Language Support

## ✅ Реализовано / Implemented

Сайт теперь поддерживает **три языка**:
- 🇬🇧 **English** (Английский)
- 🇷🇺 **Русский** (Russian)
- 🇰🇿 **Қазақша** (Kazakh)

## 🎯 Как использовать / How to Use

### Переключение языка / Language Switching

1. В правом верхнем углу сайта найдите переключатель языков с флагом
2. Нажмите на него, чтобы открыть выпадающее меню
3. Выберите нужный язык из списка
4. Весь контент сайта автоматически переведется

### Что переведено / What's Translated

✅ **Header** (Шапка сайта)
- Кнопки Login/Contact
- Переключатель языков

✅ **Hero Section** (Главная секция)
- Заголовок и подзаголовок
- Кнопки CTA
- Статистика (Active Teachers, Lessons Created)

✅ **Features** (Преимущества)
- Все заголовки и описания функций

✅ **Tools Carousel** (Карусель инструментов)
- Названия и описания всех AI инструментов

✅ **Testimonials** (Отзывы)
- Заголовки секции

✅ **Pricing** (Цены)
- Названия тарифов
- Описания планов
- Кнопки

✅ **Footer** (Подвал)
- Все ссылки и разделы

## 🔧 Для разработчиков / For Developers

### Добавление нового перевода

1. Откройте `src/translations/translations.js`
2. Добавьте новый ключ во все три языка (en, ru, kk)
3. Используйте в компоненте:

```javascript
import { useTranslations } from '../hooks/useTranslations';

const MyComponent = () => {
  const { t } = useTranslations();
  
  return <div>{t('myNewKey')}</div>;
};
```

### Текущий язык по умолчанию

По умолчанию установлен **английский** (en). Язык сохраняется в состоянии компонента и меняется при выборе пользователя.

- `kk` - Қазақша
