# 🔑 Как настроить OpenAI API ключ

## 📍 Где получить ключ

1. **Зайдите на сайт OpenAI:**
   - Перейдите: https://platform.openai.com/
   - Войдите или зарегистрируйтесь

2. **Создайте API ключ:**
   - Перейдите: https://platform.openai.com/api-keys
   - Нажмите **"Create new secret key"**
   - Придумайте название (например: "Aldea Tools")
   - **СКОПИРУЙТЕ КЛЮЧ** (он начинается с `sk-...`)
   - ⚠️ Важно: Ключ показывается только один раз!

3. **Пополните баланс (если нужно):**
   - Перейдите: https://platform.openai.com/account/billing
   - Добавьте способ оплаты (минимальный депозит $5)

---

## 💻 Куда поставить ключ (локально)

1. **Создайте файл `.env` в корне проекта:**
   - Откройте папку проекта
   - Создайте файл с названием `.env` (точка в начале!)
   - Файл должен быть в той же папке, где `package.json`

2. **Вставьте в файл `.env` следующий текст:**
   ```env
   OPENAI_API_KEY=sk-ваш-ключ-здесь
   OPENAI_MODEL=gpt-4o-mini
   ```

3. **Замените `sk-ваш-ключ-здесь` на реальный ключ:**
   ```env
   OPENAI_API_KEY=sk-proj-abc123def456ghi789jkl012mno345pqr678stu901vwx234yz
   OPENAI_MODEL=gpt-4o-mini
   ```

4. **Сохраните файл**

---

## 🌐 Куда поставить ключ (Netlify)

1. **Войдите в Netlify Dashboard:**
   - Перейдите: https://app.netlify.com/
   - Выберите ваш сайт

2. **Добавьте переменные:**
   - Перейдите: **Site settings** → **Environment variables**
   - Нажмите **"Add a variable"**
   - Добавьте:
     - **Key:** `OPENAI_API_KEY`
     - **Value:** `sk-ваш-ключ-здесь` (вставьте ваш ключ)
   - Нажмите **"Save"**

3. **Добавьте модель (опционально):**
   - Нажмите **"Add a variable"** еще раз
   - Добавьте:
     - **Key:** `OPENAI_MODEL`
     - **Value:** `gpt-4o-mini`
   - Нажмите **"Save"**

4. **Перезапустите деплой:**
   - Перейдите в **Deploys**
   - Нажмите **"Trigger deploy"** → **"Clear cache and deploy site"**

---

## 🤖 Как выбрать модель

### Доступные модели:

| Модель | Цена | Качество | Рекомендация |
|--------|------|----------|--------------|
| `gpt-4o-mini` | Дешево ($0.15/1M токенов) | Хорошее | ✅ **Рекомендуется для начала** |
| `gpt-4o` | Дорого ($2.50/1M токенов) | Отличное | Для премиум функций |
| `gpt-4-turbo` | Очень дорого ($10/1M токенов) | Отличное | Альтернатива gpt-4o |

### Как изменить модель:

**Для локальной разработки:**
- Откройте файл `.env`
- Измените строку `OPENAI_MODEL=gpt-4o-mini` на нужную модель
- Сохраните и перезапустите сервер

**Для Netlify:**
- Перейдите в **Site settings** → **Environment variables**
- Найдите `OPENAI_MODEL` или создайте новую переменную
- Измените значение на нужную модель
- Сохраните и перезапустите деплой

## ✅ Проверка

После настройки:

1. **Запустите сервер:**
   ```bash
   npm run server
   ```

2. **Попробуйте создать контент:**
   - Откройте любой инструмент (например, "План урока")
   - Введите тему
   - Нажмите "Создать"
   - Должен появиться сгенерированный контент

## 🔒 Безопасность

⚠️ **НИКОГДА не публикуйте ваш API ключ!**
- ✅ Файл `.env` уже добавлен в `.gitignore`
- ❌ Не коммитьте ключ в код
- ❌ Не делитесь ключом публично

---

## 🆘 Проблемы?

### Ошибка: "OPENAI_API_KEY is not configured"
- Проверьте, что файл `.env` создан
- Убедитесь, что ключ введен правильно
- Перезапустите сервер

### Ошибка: "Invalid API Key"
- Проверьте, что ключ скопирован полностью
- Убедитесь, что нет лишних пробелов
- Проверьте баланс на OpenAI
